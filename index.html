Conceptos Clave del Trimestre 1: Diseño y Desarrollo Web

GitHub

GitHub es una plataforma de alojamiento de código para el control de versiones y la colaboración. Utiliza Git, un sistema de control de versiones distribuido, que permite a los desarrolladores:

  • Almacenar y gestionar sus proyectos de software.
  • Controlar los cambios realizados en el código a lo largo del tiempo.
  • Colaborar eficientemente con otros desarrolladores en el mismo proyecto, fusionando sus contribuciones sin conflictos.
  • Volver a versiones anteriores del código si es necesario.

Es ampliamente utilizado en el desarrollo web para la gestión de proyectos, desde pequeños sitios personales hasta grandes aplicaciones empresariales.

Hosting

El hosting (o alojamiento web) es el servicio que permite a una organización o individuo hacer que su sitio web o aplicación web sea accesible en Internet. Cuando contratas un servicio de hosting, un proveedor te asigna espacio en un servidor web donde se almacenan todos los archivos (HTML, CSS, JavaScript, imágenes, etc.) y bases de datos necesarios para tu sitio.

Las características clave del hosting incluyen:

  • Almacenamiento: Espacio en el servidor para guardar los archivos de tu sitio.
  • Ancho de banda: La cantidad de datos que pueden ser transferidos desde el servidor a los visitantes de tu sitio.
  • Tiempo de actividad (Uptime): La garantía de que tu sitio estará en línea y accesible la mayor parte del tiempo.
  • Soporte técnico: Ayuda del proveedor para solucionar problemas.

Sin hosting, tu sitio web no podría ser visto por nadie en línea.
